Overview on C++

C++ is a cross-platform language that can be used to create high performance applications and solve complex problems. The language was developed by Bjarne Stroustrup at Bell Labortories over a period starting in 1979. It was created as an extension to the C programming language. The language puts a higher level of control over the system resources and memory in the developers hands.

C++ is also one of the world's most popular programming languages. The programming language can be found in many of today's operating systems, Graphical User Interfaces (GUI), and embedded systems. C++ is an object-oriented programming language which applies a structural approach to programming and lends itself well to allowing code to be reused, which helps in lowering development costs. The language is also a portable language, meaning that it can be used to develop applications for multiple platforms. No wonder why it's so popular!
