Overview on HTML and CSS

So you want to create your own website? Good! Before you enroll in this course, it's important that you know some history on HTML and CSS.

The first version of HTML was written by Tim Berners-Lee in 1993 and would go on to have numerous iterations after with the most widely used version in the 2000s being HTML 4.01. HTML 4.01 became the official standard in December 1999. There was another version called XHTML which was a rewrite of HTML as a standard markup language. Its pretty similar but employs stricter rules on syntax. It became an official standard in 2000, and was updated in 2002. Most web pages today use either HTML 4.01 or XHTML 1.0. We are currently on HTML 5 the newest version of HTML created with the collaboration of the World Wide Web Consortium (W3C) and the Web Hypertext Application Technology Working Group (WHATWG).

So what exactly is HTML? Well, it stands for Hyper Text Markup Language and is used for creating web pages. It does this by describing the structure of the web page content and consists of a series of elements, such has "header, body, and footer." These elements tell your web browser how to display the content in the file! Now, what about CSS? CSS was first proposed by Håkon Wium Lie in 1994. CSS stands for Cascading Style Sheets and works hand in hand with HTML. CSS is used to describe "HOW" the HTML elements are displayed on the screen, essentially giving the web page "style." It can control the layout of multiple web pages at once saving time and promoting reusability!
